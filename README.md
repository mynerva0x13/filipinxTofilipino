# Filipinx → Filipino Browser Extension

A simple browser extension that automatically replaces the word **“Filipinx”** with **“Filipino”** across websites.

## ✨ Features
- Replaces all instances of **“filipinx”** (case-insensitive) with **“filipino”**
- Works in real-time on most websites
- Lightweight and fast
- Designed to support consistent terminology

## 🚀 Version
**v1.0.0**
- Initial release
- Basic text replacement functionality
- Intended to work across all major websites (may vary depending on site structure)

## 🧩 Installation

### Firefox
- Install directly from the Firefox Add-ons store:  
👉 https://addons.mozilla.org/en-US/firefox/addon/filipinx-to-filipino-replacer/

### Chrome / Chromium-based Browsers
1. Download the `.crx` file from the repository

2. Enable Developer Mode:
   - Go to `chrome://extensions/`
   - Toggle **Developer Mode** (top right)

3. Install:
   - Drag and drop the `.crx` file into the extensions page  
   **OR**
   - Load unpacked extension if source files are provided

## ⚙️ How It Works
The extension scans webpage text and replaces occurrences of the word **“filipinx”** with **“filipino”** dynamically using content scripts.

## ⚠️ Notes
- Some websites with heavy scripts or dynamic rendering (e.g., React apps) may not fully update all instances.
- Future updates will improve compatibility across all sites.
